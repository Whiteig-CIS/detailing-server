Iggy's Detailing API

An open endpoint serving detailing packages for boats and automobiles.

Available Endpoints

  • GET /api/packages — Returns a JSON array of all detailing packages.

View raw JSON: /api/packages

Example Usage

Fetch with JavaScript:

fetch("/api/packages")
  .then(res => res.json())
  .then(data => console.log(data));

Response Structure

Each package object includes:

  • _id — unique identifier
  • vehicle_type — "auto" | "boat"
  • teir — "bronze" | "silver" | "gold"
  • interior_services — string[]
  • exterior_services — string[]
  • starting_price — number
  • summary — description
  • preview_image — filename
